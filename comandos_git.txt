grgit init . -> cria um repo git na pasta atual
git status -> mostra o status do trabalho (de cada arquivo)
git add nome.arquivo -> adiciona o arquivo a ser commitado
git commit -m "msg" -> faz commit do que estava pra ser commitado
git log -> mostra todo historico de commit 

FLUXO DE TRABALHO
1. Criar o diretório (pasta) do projeto
2. Iniciar o git dentro do diretório criado
3. Edita ou cria arquivos
4. Git status
5. Git add <nome arquivo>
6. Git status
7. Git commit -m "mensagem"
8. Git status
9. Volta para o passo 3
